Stock Management Web App

Built with Vaadin, Spring Security/Data and Hibernate. Its responsive, has authorization and authentication for users, and CRUD full functionalities for items.

A desktop inventory management application built with Java, MySql and Gluon with fxml files for the GUI. With account creation, CRUD fuctionality for items.
